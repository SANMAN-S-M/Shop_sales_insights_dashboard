# Shop_sales_insights_dashboard
This project presents an interactive sales dashboard built using Microsoft Excel. It provides insights into sales performance, order status, customer demographics, and channel distribution.


## 📁 Files Included
Dashboard.xlsx → Complete Excel file with data, pivot tables, and dashboard
Report.png → Preview of dashboard

## 📊 Key Features
Orders vs Sales trend analysis
Order status breakdown
Sales by gender
Top 5 states by sales
Age vs Gender analysis
Channel-wise distribution

## 🛠 Tools Used
Microsoft Excel
Pivot Tables
Pivot Charts
Slicers
